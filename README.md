# 👩🏻‍💻 Geremias Santos

**`Analista de Segurança do Produto especializado em análise de dados para investigação de fraude.`**

Sou Geremias Santos, tenho 31 anos e moro em São Paulo (SP). Sou formado em Análise e Desenvolvimento de Sistemas e atualmente curso pós-graduação em Engenharia de Machine Learning pela FIAP.

Atuo na área de dados com foco em análise, investigação e desenvolvimento de soluções voltadas à prevenção de fraudes. Tenho experiência no desenvolvimento de ferramentas investigativas, além de sólida atuação em análise geoespacial e análise de grafos. Também trabalho com a construção de pipelines de dados e utilizo Python e SQL para transformar grandes volumes de informação em inteligência aplicada à tomada de decisão.
